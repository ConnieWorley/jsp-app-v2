What changed: The tailoring process is no longer reconstructed from memory. A structured checklist is generated automatically on JD save — default items plus the specific skills gaps surfaced in Stage 1. Due dates are auto-calculated from prior item completion using offsets configured in Setup; the user can edit any date. The AI Resume Coach provides targeted guidance in context; the job seeker makes the edits. Status update to Applied triggers Stage 3.
