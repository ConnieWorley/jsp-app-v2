What changed: Setup is no longer a blank page. The app provides a structured wizard with guided questions, progress tracking, and explicit gating — required sections must be complete before the job search stages unlock. The Follow-Up Schedule is a required setup section: the user configures offset values once (e.g., tailor resume within 1 day, initiate networking within 2 days, follow up within 4 days of outreach), and the app uses these to auto-calculate due dates across all stages based on prior item completion timestamps. All data is stored once and reused downstream, eliminating the scattered-tools problem entirely.
