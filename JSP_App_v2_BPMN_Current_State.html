Key finding: The current process has tracking infrastructure but no methodology. Every tailoring and prep step is reconstructed from memory each time. Information is scattered across 5+ disconnected tools with no connective tissue between them. Networking and interview prep are triggered inconsistently or not at all. The problem is process, not software — and that is exactly what JSP App v2 is designed to fix.
